# Równania różniczkowe i różnicowe - zadanie obliczeniowe

Zgodnie z algorytmem losowania, projekt realizuje wariant 5: potencjał elektromagnetyczny.

```
mappedValue = ord('T') - ord('A')   # 19
variant = mappedValue % 5 + 1       # 5
```

## Tech stack:
Projekt zrealizowano przy pomocy **TypeScript**, front-end został zbudowany przy użyciu **React**. Do wykresów użyto **Recharts**, a do renderowania *LaTeX* **react-katex**.
